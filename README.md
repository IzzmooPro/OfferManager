### 🚀 Kurulum

1. [**Son Sürümü indirin**](https://github.com/IzzmooPro/OfferManager/releases/latest) → `TeklifYonetim_Setup_v<sürüm>.exe`
2. Kurulumu çalıştırın, program açılsın. Hepsi bu! 🎉

### 🔄 Güncellemeler Nasıl Çalışır?

Program her açılışta yeni sürüm olup olmadığını kontrol eder. Varsa **size sorar** — indirir, onayınızla kurar. Zorla güncelleme yok; her adımda onay istenir. Güncelleme yalnızca programı yeniler, **verilerinize dokunmaz**.

### 🔒 Verileriniz Güvende

- 📁 Veriler → `%LOCALAPPDATA%\OfferManagementSystem\data`
- 🗄️ Yedekler → `Belgeler\OfferManagementSystem\backups`

Bu klasörler programdan ayrıdır; güncelleme ve yeniden kurulumdan **etkilenmez**.

### 🛠️ Kaynaktan Çalıştırma (Geliştiriciler)

```bash
pip install -r requirements.txt
python main.py
```

> Python 3.12+ gerekir. Program eksik paketleri kendisi indirmez; eksik varsa hangi paketin gerektiğini bildirip durur.

## 🇬🇧 English

> **Pick a product, pick a customer, create the quote — let the app do the rest.**

Save your products and customers, build a professional quotation with a 3-step wizard, and export it as a **PDF** in one click. All your data stays safely on your own computer.

### ✨ Highlights

| | Feature |
|---|---|
| 🧙 | **3-step quote wizard** — customer → products → terms → ready quote |
| 📄 | **Professional PDF** — with logo, signatures, full Turkish support |
| 💰 | **Profit analysis** — while building a quote, see your profit and the **maximum discount** you can offer based on cost — **visible only to you**, never shown to the customer or in the PDF |
| 👥 | **Customer & product management** — search, filter, multi-select & bulk delete, Excel/CSV import-export (handles thousands of records) |
| 🗂️ | **Categories & templates** — reuse frequent quotes in one click |
| 📊 | **Reports** — monthly revenue, customer/product rankings, conversion rate |
| 💱 | **Multi-currency** — ₺ TRY · € EUR · $ USD |
| 🌗 | **Light / Dark theme** |
| 💾 | **Automatic backups** and restore |
| 🔔 | **Auto-update** — the app notifies you when a new version is out |

### 🚀 Installation

1. [**Download the latest release**](https://github.com/IzzmooPro/OfferManager/releases/latest) → `TeklifYonetim_Setup_v<version>.exe`
2. Run the installer and launch the app. That's it! 🎉

### 🔄 How Updates Work

On startup the app checks for a newer version. If there is one, it **asks you first** — downloads it, and installs on your confirmation. No forced updates, every step needs approval. Updates only refresh the program and **never touch your data**.

### 🔒 Your Data Is Safe

- 📁 Data → `%LOCALAPPDATA%\OfferManagementSystem\data`
- 🗄️ Backups → `Documents\OfferManagementSystem\backups`

These folders are separate from the program and are **untouched** by updates or reinstalls.

### 🛠️ Run From Source (Developers)

```bash
pip install -r requirements.txt
python main.py
```

> Requires Python 3.12+. The app does not download missing packages by itself; if something is missing it reports which package is required and stops.
